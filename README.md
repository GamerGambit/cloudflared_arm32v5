# Cloudflared Arm32v5

## Disclaimer
The image produced by this project includes the official unmodified `cloudflared` binary built by Cloudflare and provided under the Apache License, Version 2.0.

This image is a community contribution and is not affiliated with `cloudflared` or Cloudflare.

## Introduction
Docker image of `cloudflared` built for arm32v5 specifically for use on Mikrotik devices running the EN7562CT CPU (e.g. hEX 2024/2025).
